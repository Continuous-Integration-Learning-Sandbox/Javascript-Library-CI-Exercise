# Contributing

This is a learning sandbox for a Continuous Integration course.

Contributions are welcome mainly for educational purposes.

## How to contribute:
1. **Fork** the repository to your own account.
2. **Create a branch** for your changes.
3. **Make your changes** and ensure the tests still pass (**npm test**).
4. **Submit a Pull Request** with a brief description of what you did.

Please make sure your code is clean and follows the existing style of the project.
