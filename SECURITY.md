# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within this package, please do not open a public issue. 

Instead, please report it directly through the npm registry:
1. Go to the package page on [npmjs.com](https://www.npmjs.com).
2. Use the "Report malware" or "Report vulnerability" link located in the sidebar.

This is an educational project. No active support or security updates are guaranteed.
